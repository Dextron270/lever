As computing power and capabilities increase, even very complex computer programs can be written in smooth, simple and expressive languages.

Rapid language development tools, fully customizable syntax and semi-automatic C-FFI in lever ensures you're not running out of options. Dynamically typed runtime and language ensure there are no mathematical straitjackets slowing down your keystrokes. Code off like a madman!

  • RPython implemented runtime.
  • Built-in eventloop with powerful concurrent programming framework based on greenlets.
  • Plugin-architectures and hot-reloadable code with module scopes.
  • Intuitive lexical scoping, operator overloading via multimethods.
  • GLSL-style runtime-supported vector, matrix and quaternion math.
  • Growing library support for computer game developers, virtual reality, audio and 3D graphics.
  • Partial Windows, SteamOS development support.
  • Partial out-of-the-box OpenGL 4 and Vulkan support.
  • MIT license.
