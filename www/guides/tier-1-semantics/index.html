Tier 1 - Structure of Lever programs

Awareness of Lever should come up in Tiers that are guided by our curriculum. Skills in one category are not useful without the others.

So I've started to draft tiers that each teach harder topics than earlier one from each category.

This guide would teach the first tier of:

  • Knowledge of the Lever programming language
  • Writing instructions in Lever language

Considering what this guide should contain: functions variables, function scope? elif/else? while? for? Decided that it will at least contain: basic program flow / evaluation rules. local variables if rules for equality and inequality ==, != false/null -condition.
